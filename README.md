# algorithms
This will contain coded examples of basic algorithms, each of which are contained within a single .py file that fits their type.

basic_searches.py contains:
  - Linear search
  - Binary search

basic_sorts.py contains:
  - Bubble sort
  - Insertion sort
  - Merge sort
  - Quick sort
  
ciphers.py contains:
  - Affine Cipher
  - Atbash Cipher
  - Baconian Cipher
  - Ceaser Cipher
  - Hill Cipher *
  - Key Phrase Cipher *
  - Morse Code
  - Rail Fence Cipher *
  - ROT13 Cipher
  - Substitution Cipher

Note: Those with a * are not yet fully complete.
